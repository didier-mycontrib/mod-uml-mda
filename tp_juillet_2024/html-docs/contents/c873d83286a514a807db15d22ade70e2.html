enregistrer début empunt

UMLUseCase
bibliotheque :: ModelUseCase :: enregistrer début empunt

Description

scénario nominal:

  1. renseigner (par saisie ou lecture code) le numéro d'abonné
  2. vérifier droits à emprunter (voir sous U.C)
  3. préciser le numéro l'exemplaire à emprunter
  4. enregistrer tous les détails de l'emprunt (emprunteur , date , ....)
  5. afficher résumé ou confirmation sur emprunt effectué
