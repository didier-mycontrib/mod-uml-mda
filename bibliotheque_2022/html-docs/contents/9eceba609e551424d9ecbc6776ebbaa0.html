UMLUseCase

Description

pré-condition:

abonné préalablement identifié

scénario nominal (vérifier droits à emprunter):

  1. vérifier abonnement valide (pas expiré)
  2. vérifier absence de contentieux
  3. vérifier nombre d'exemplaires déjà empruntés et pas encore rendus <= max_autorisé
  4. remonter exception ou bien nombre d'exemplaires autorisés à être empruntés
